欢迎来到我的博客

发布于 2025-11-21

这是使用Hexo搭建的博客演示页面。Hexo是一个快速、简洁且高效的博客框架，基于Node.js开发。

Hexo使用Markdown解析文章，可以快速生成静态文件，并且有丰富的主题和插件系统。

主要特性包括：

  • 支持GitHub Flavored Markdown的所有功能
  • 一键部署到GitHub Pages, Heroku等平台
  • 强大的插件系统
  • 活跃的社区支持

如何搭建Hexo博客

发布于 2025-11-20

搭建Hexo博客非常简单，只需要几个步骤：

  1. 安装Node.js和npm
  2. 全局安装Hexo CLI: npm install -g hexo-cli
  3. 初始化项目: hexo init blog
  4. 进入目录并安装依赖: cd blog && npm install
  5. 启动服务器: hexo server
