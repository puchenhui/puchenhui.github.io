# GitHub Portfolio & Blog

This is a personal portfolio and blog that can serve as your GitHub profile page. 

## How to Set Up Your GitHub Profile Page

1. Create a new repository with the name `your-username.github.io` (replace `your-username` with your actual GitHub username)
2. Push this code to that repository
3. Enable GitHub Pages in your repository settings:
   - Go to Settings > Pages
   - Select "Deploy from a branch"
   - Choose "main" branch
   - Click "Save"

## Customization

To customize this portfolio for yourself:

1. Edit `index.html` to update your name, bio, projects, and blog posts
2. Replace `profile.jpg` with your own profile image (or update the image source)
3. Update social media links in the footer
4. Modify colors in `styles.css` to match your preferences

## Features

- Responsive design that works on all devices
- Clean, modern UI with smooth hover effects
- Sections for About, Projects, and Blog
- Social media links in the footer
- Professional typography and spacing
